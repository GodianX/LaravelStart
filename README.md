# LaravelStart

### System requirements

- Docker 

### Installation

- `cd docker`
- `docker-compose up -d --build`
- `docker exec -it laravel_php bash`
- `cd ../laravelStart/`
- `composer install`

### Web

- `http://127.0.0.1:1080/`

- password: `laravel`
- database: `postgres`
